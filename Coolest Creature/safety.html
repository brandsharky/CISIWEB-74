Web Safety

Using the web is amazing. It makes things easier and helps you finish tasks a lot faster. However, the web can be dangerous. Follow these tips on web safety and security to help you get the safest experience on the web.

When using the Internet, your computer stores information. However, web apps can access some of this information. Web apps have access to the following:

Properties Description
appName Name of the web browser showing the page
appVersion Version of the web browser showing the page
geolocation API for accessing the user's current location and user permission settings denying or allowing access to that information
platform Operating system being used on the client computer
userAgent String stored in the HTTP user-agent request header, which contains information about the browser, the platform name, and compatibility
pixelDepth Display screen's color resolution in bits per pixel
colorDepth Display screen's bit depth if a color palette is in use; if a color palette is not in use, returns the value of the pixelDepth property
